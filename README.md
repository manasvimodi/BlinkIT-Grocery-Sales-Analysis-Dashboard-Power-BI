# BlinkIT Grocery Sales Analysis Dashboard | Power BI

This project is an interactive **Power BI dashboard** built using the BlinkIT Grocery dataset. It provides meaningful business insights into sales performance, outlet distribution, product categories, and customer ratings through dynamic visualizations and KPIs.

## Dashboard Highlights

* Total Sales Analysis
* Sales by Outlet Type
* Sales by Outlet Size
* Sales by Outlet Location
* Sales Trend by Establishment Year
* Sales by Item Type
* Item Fat Content Analysis
* Average Customer Rating
* Interactive Filters and Slicers

## Tools Used

* Power BI
* Microsoft Excel
* Power Query
* DAX

## Dataset

The dataset contains grocery sales information, including:

* Item Type
* Fat Content
* Outlet Type
* Outlet Size
* Outlet Location
* Establishment Year
* Sales
* Rating

## Key Insights

* Identified top-performing outlet types and locations.
* Analyzed the impact of outlet size on sales.
* Compared sales across different item categories.
* Evaluated customer ratings alongside sales performance.
* Created an interactive dashboard for quick business decision-making.
